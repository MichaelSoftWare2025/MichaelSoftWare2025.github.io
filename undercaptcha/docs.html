UnCaptcha.js Documentation

UnCaptcha.js - это лёгкий скрипт для создания и проверки текстовой капчи.

1. Подключение

Добавьте следующий код в ваш HTML-файл:

<script src="MichaelSoftWare2025.github.io/undercaptcha/uncaptcha.min.js"></script>

2. Использование

Просто добавьте следующий HTML-код на вашу страницу:

<div class="uncaptcha" url="https://example.com"></div>

Атрибут url указывает, куда будет перенаправлен пользователь после успешного ввода капчи.

3. Как это работает?

После загрузки страницы UnCaptcha.js автоматически:

  • Находит все элементы div.uncaptcha.
  • Генерирует случайный текст капчи.
  • Рисует капчу с небольшими наклонами символов.
  • Добавляет поле ввода и кнопку подтверждения.

4. Функции

  • generateCaptcha() - создаёт случайную строку капчи.
  • generateCaptchaImage(text) - рендерит изображение капчи на основе текста.
  • createCaptchaElement(container) - создаёт капчу внутри переданного элемента.
